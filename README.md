# evgeshayoga

A cross-platform mobile app for displaying video content from evgeshayoga.com Yoga Portal

## Tech

- Flutter
- Firebase (Auth, Database) + REST
- Chewie video player


## Getting Started

Following components are required
 - Flutter ❤️ (^1.9.0)
 - Xcode
 - Android Studio
 - cocoapods (^1.8.0)
